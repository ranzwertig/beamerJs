example markup

Your minimal body markup should look like this:

<body>
	<header>
		<h1 id="title"></h1>
		<h2 id="subtitle"></h2>
	</header>
	<section class="slide">
		<h1 class="title">the slides title</h1>
		<h2 class="subtitle">some subtitle</h2>
	</section>
	<footer>
		some footer content
	</footer>
</body>

beamerJs

markup description

  • header: The header is displayed on the top of each slide.
    The title and the subtitle of each slide is displayed in the headers h1 and h2 tag.
  • footer: The footer is always shown at the end of a slide, there you can provide information about the the author...
  • <section class="slide">: Each slide is defined by it's own <section clsss="slide">. It is possible to add a title and a subtitle which are displayed in the header.
    • titelslide: If you add the class titelslide to the slide section the title and subtitle are displayed centered on the slide.
    • tableofcontents: A slide which displays the TOC of the whole presentation.
